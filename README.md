## 复数
下面是复数功能的界面

![复数的界面](./image/复数/复数的界面.png)

在输入形式处，你可以选择你的输入形式，分为直角坐标和极坐标两种输入方式，在复选框，你可以选择计算加法、减法、乘法或是除法

最后的两行是计算的结果，以两种形式分别呈现

记住，你做的任何更改，都会引起计算结果的改变
